# BRAC University CS Core Cheatsheets

This repository contains concise cheatsheets for all **core Computer Science courses** offered in the **BRAC University CS Undergraduate Program.** 
These materials were originally prepared for personal academic revision and faculty recruitment preparation (**Lecturer positions in private universities**), but may also benefit current students, job applicants, and educators.
